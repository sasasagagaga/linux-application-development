# Guesser documentation

Guesser -- a program that guesses your number.

## Usage
guesser [OPTION...]

### Options
* `-r, --use_roman` -            Use roman digits, not arabic
* `-?, --help` -                 Give this help list
* `-u  --usage` -                Give a short usage message
* `-V, --version` -              Print program version
